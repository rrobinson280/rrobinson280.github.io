A VISUAL INTUITION FOR SPACE FILLING CURVES

RYAN ROBINSON
A space-filling curve can be thought of as a continuous mapping from a lower dimensional space to a higher one. This article attempts to provide visual intuition for this mapping as well as rigorous proof.

Let I = [0, 1]. There exists a continuous map f : I → I2 whose image fills up the entire square I2.

Useful Theorems

Theorem 1 Let X be a topological space and let (Y,d) be a metric space. The set (X,Y ) of continuous functions is closed in Y X under the uniform metric. So is the set (X,Y ) of bounded functions. Therefore, if Y is complete, these spaces are complete in the uniform metric.

Uniform Limit Theorem. Let fn : X → Y be a sequence of continuous functions from the topological space X to the metric space Y . If (fn) converges uniformly to f, then f is continuous.
